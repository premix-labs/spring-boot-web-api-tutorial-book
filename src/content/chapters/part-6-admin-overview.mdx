# ภาค 6: Admin System

ภาคนี้ต่อยอดจาก login/register ไปสู่ระบบ admin ที่มี role, permission, admin-only endpoints, user management และ audit log

หลังจบภาคนี้ โปรเจกต์ `Backend API` จะมีโครงสร้างระบบผู้ดูแลที่ใช้ได้จริงมากขึ้น และพร้อมเข้าสู่ภาค production-ready
