## หนังสือเล่มนี้สอนอะไร

- **Foundation**: พื้นฐาน Spring Boot
- **REST API**: controller และ CRUD
- **Database**: PostgreSQL และ JPA
- **Auth**: register, login และ JWT
- **Admin**: role และ endpoint ผู้ดูแล
- **Production**: test, Docker และ Swagger

## โปรเจกต์หลักของหนังสือ

เราจะสร้าง `Backend API` ทีละขั้น จาก endpoint ธรรมดาไปถึง backend ที่มีผู้ใช้ 2 กลุ่ม:

- `USER`: สมัคร, login, ดูข้อมูลตัวเอง
- `ADMIN`: จัดการผู้ใช้และ audit log

เป้าหมายไม่ใช่แค่ให้โค้ดรันได้ แต่ให้เข้าใจว่าโครงสร้างแบบนี้นำไปต่อยอดงานจริงได้อย่างไร
