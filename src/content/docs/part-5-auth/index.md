---
title: "ภาค 5: Login/Register"
description: เพิ่ม Authentication ด้วย Spring Security และ JWT
---
ภาคนี้เริ่มเข้าสู่หัวข้อที่เจอในงานจริงแทบทุกระบบ: การสมัครสมาชิก, เข้าสู่ระบบ, hash password, ออก JWT, อ่าน current user และป้องกัน endpoint ด้วย Spring Security

หลังจบภาคนี้ โปรเจกต์ `SecureAdmin API` จะมี authentication flow ที่ต่อยอดไปสู่ระบบ admin และ role-based access control ได้
