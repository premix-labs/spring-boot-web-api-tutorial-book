description: ปูพื้นฐาน backend, Spring Boot และเครื่องมือที่ต้องใช้
---
ภาคนี้ทำให้ผู้อ่านเข้าใจภาพรวมก่อนเริ่มเขียนโค้ดจริง:

- Spring Boot คืออะไร
- Backend และ REST API ทำงานอย่างไร
- ต้องติดตั้งเครื่องมืออะไรบ้าง
- โครงสร้างโปรเจกต์ Spring Boot อ่านอย่างไร

ผลลัพธ์ปลายภาคคือผู้อ่านควรสร้างและรัน Spring Boot project แรกได้
