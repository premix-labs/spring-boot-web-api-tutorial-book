---
title: "ภาค 3: ฐานข้อมูล"
description: เชื่อมต่อ PostgreSQL และใช้ Spring Data JPA
---
ภาคนี้เปลี่ยนจาก in-memory data ไปใช้ PostgreSQL จริง เริ่มจากการตั้งค่า datasource แล้วค่อยสร้าง JPA entity, repository, CRUD service/controller และ update endpoint

หลังจบภาคนี้ ผู้อ่านจะมี User API ที่บันทึกข้อมูลลง database จริง และพร้อมนำไปปรับปรุงต่อในภาค Better API
