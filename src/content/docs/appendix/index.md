---
title: ภาคผนวก
description: Cheat sheet และ reference สำหรับเปิดดูเร็ว
---
ภาคผนวกเก็บข้อมูลที่ผู้อ่านควรเปิดดูซ้ำระหว่างทำโปรเจกต์

หน้าแนะนำ:

- `Final Project`: วิธีเปิดดูและรัน `examples/final-backend-api/`
- `GitHub Pages Deploy`: วิธี deploy เว็บหนังสือขึ้น GitHub Pages
- `Gradle Commands`: คำสั่ง Gradle ที่ใช้บ่อย
- `Postman Collection`: แนวทางจัดชุด request สำหรับทดสอบ API
