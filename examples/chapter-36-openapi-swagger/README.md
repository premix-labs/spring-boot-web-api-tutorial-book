# Chapter 36 - OpenAPI และ Swagger

ตัวอย่างนี้ใช้ประกอบบทที่ 36 สำหรับเพิ่ม OpenAPI และ Swagger UI

ไฟล์ที่ใช้:

- `build.gradle-snippet.gradle`
- `OpenApiConfig.java`
- `AuthController.java`

สำหรับ Spring Boot 4 ตัวอย่างนี้ใช้ `springdoc-openapi-starter-webmvc-ui:3.0.3`

Swagger UI:

```text
http://localhost:8080/swagger-ui/index.html
```
