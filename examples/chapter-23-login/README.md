# Chapter 23 - Login

ตัวอย่างนี้ใช้ประกอบบทที่ 23 สำหรับเพิ่ม login endpoint และตรวจ password ด้วย `PasswordEncoder`

ไฟล์ที่ใช้:

- `LoginRequest.java`
- `LoginResponse.java`
- `AuthController.java`
- `AuthService.java`
- `GlobalExceptionHandler.java`
- `requests.http`

บทนี้ยังคืน token placeholder ก่อน บทที่ 24 จะเปลี่ยนเป็น JWT จริง
