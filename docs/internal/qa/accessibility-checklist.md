# Accessibility Checklist

ใช้ checklist นี้กับเว็บไซต์หนังสือก่อนเผยแพร่

## Navigation

- ใช้ keyboard ไปยัง sidebar, search และ content ได้
- focus state มองเห็นได้
- link text สื่อความหมาย
- heading hierarchy ไม่กระโดดผิดลำดับจนอ่านยาก

## Content

- code block อ่านได้และไม่บัง content
- table มี header ชัดเจน
- callout/warning ไม่ใช้สีอย่างเดียวเพื่อสื่อความหมาย
- diagram มีคำอธิบายประกอบในเนื้อหา

## Mobile

- sidebar ใช้งานได้
- search ใช้งานได้
- code block scroll ได้
- table ไม่ทำให้หน้าจอแตก
